# Api Cafeto

## Description
Api Rest where you can manage events and users

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# watch mode
$ npm run start:dev:watch

# production mode
$ npm run start
```

## Build the app
```bash
$ npm run build
```
